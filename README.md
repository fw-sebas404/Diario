# 📖 Mi Diario Personal

Aplicación web de diario personal con clave de seguridad y almacenamiento en la nube.

## Características

- **Clave de seguridad** para proteger el acceso al diario
- **Editor de entradas** con título y contenido libre
- **Guardado automático** en la nube (GitHub) cada vez que escribes
- **Sincronización** entre dispositivos: lo que escribes en un lugar aparece en todos
- **Búsqueda** de entradas por título o contenido
- **Diseño oscuro** elegante y responsive (funciona en móvil y escritorio)

## Acceso

La aplicación está disponible en:  
**https://fw-sebas404.github.io/Diario/**

## Almacenamiento

Todas las entradas se guardan en el archivo `data.json` de este repositorio, lo que garantiza que los datos persistan en la nube y estén disponibles desde cualquier dispositivo.
